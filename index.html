Website Guide

This website showcases the physics of a two gyroscope car lift using the moving frame method.

Simulation Controls

One can control which case of the simulation one wants to display in two ways: Use the "Case" drop down button to the right of the screen, or press the right mouseclick button.

To the right of the screen, there is a TweakPane GUI which contains 4 folders:

Simulation Controls: Here can the user control different parameters of the simulation such as:

Case: This drop-down button allows the user to switch between different simulation cases.

simSpeed: This slider lets the user control the simulation speed.

cameraAngle: This allows the user to stop any automatic camera angle swithes

ExplodedView: Shows an exploded view of the assembly with frames.

Simulation Results

The next folder 'Simulation Results' showcases the pitch and yaw angular velocities while the simulation is running.
